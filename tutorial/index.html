Sobre

Projeto realizado para a disciplina de Computação gráfica, como forma de avaliação parcial. Neste tutorial, será abordado a instalação das bibliotecas necessárias para utilização e a disponibilização do projeto desenvolvido para apresentação. A ferramenta PaperJS fornece uma manipulação grafica vetorial, de forma que as possibilidades de criação interativas se tornam "ilimitadas", como podemos ver sua capacidade em seus exemplos.

Para este projeto, foi utilizado somente a funcionalidade de curvas de bezier, que já é calculado nativamente com seus atributos handleIn e handleOut.



Instalação


PaperJS

Documentação

Paper.js — The Swiss Army Knife of Vector Graphics Scripting. Paper.js is an open source vector graphics scripting framework that runs on top of the HTML5 Canvas. It offers a clean Scene Graph / Document Object Model and a lot of powerful functionality to create and work with vector graphics and bezier curves, all neatly wrapped up in a well designed, consistent and clean programming interface.

Realizar o download dos arquivos paperjs e extrair na pasta do projeto.

Também é possível realizar a instalação através do comando: npm install paper

Adicionar o seguinte código à tag <head> da página, substituindo ___PASTA_INSTALACAO___ pelo caminho do arquivo paper-full.js.

<script src="./___PASTA_INSTALACAO___/dist/paper-full.js"></script>

Utilização

Para utilizar, basta seguir a sintaxe paper utilizando o código a seguir:

<script type="text/paperscript" src="./__caminho_pasta__/PAPER_SCRIPT_PERSONALIZADO.js" canvas="meu_canvas"></script>

Nesta sintaxe, utilizada neste projeto, é necessário criar uma tag <canvas id="meu_canvas"</canvas> com atributo id para que seja reconhecido pelo framework.

O Type type/papersript é necessário para que seja processado pelo framework e realizado a compilação do código para o equivalente em javascript puro.

Este framework trabalha com interoperabilidade com javascript, onde o código javascript é compreendido normalmente pelo framework.
